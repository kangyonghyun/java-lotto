**- 요구사항**
1. 빈 문자열 또는 null 값을 입력할 경우 0을 반환해야 한다
2. 문자열을 쉼표(,) or 콜론(:) 구분자를 기준으로 분리
3. 분리한 숫자들의 합을 반환
4. 앞에 구분자 외에 커스텀 구분자를 지정 : "//;\n1;2;3" => ; 구분자로 분리하여 합 6 반환
5. 숫자 이외에 값 or 음수를 전달 할 경우 : RuntimeException 예외를 throw
